## 🏗️ Project 1: Responsive Frontend Interface

### Overview
Project 1 focuses entirely on the foundational interface phase of full-stack development. The objective was to design and build a highly adaptable landing page from scratch using **pure, semantic frontend technologies** without relying on external CSS frameworks or libraries.

The core goal of this milestone was to respect the **Mobile-First paradigm**, ensuring that the digital entry point transitions seamlessly across smartphones, tablets, and desktop computers.

### 🌟 Key Features
* **Semantic Layout:** Built using meaningful HTML5 structural elements (`<header>`, `<nav>`, `<main>`, `<aside>`, and `<footer>`) to promote accessibility and SEO compatibility.
* **Macro & Micro Layouts:** Utilized a 2D layout framework with **CSS Grid** for high-level structural sections, paired with **Flexbox** to align micro-components like navigation links and card features.
* **2025 Aesthetic Design Tokens:** Styled intentionally with a warm, grounded color palette consisting of:
  * Mocha Mousse (`#A5856F`)
  * Ethereal Blue (`#A0D4E0`)
  * Moonlit Grey (`#F2F0EA`)
* **Responsive Architecture:** Built with `min-width` media queries to expand text layouts and grids smoothly as screen boundaries scale.
* **Fluid Typography:** Implemented `clamp()` logic so text elements resize gracefully across various screen breakpoints.
* **Interactive Logic:** Integrated vanilla JavaScript to handle a mobile burger navigation layout and simple state events.

### 🛠️ Core Toolkit Used
1. **HTML5:** Architectural landmarks and meaningful text tags.
2. **CSS3:** Flexible positioning, custom variables, and design rules.
3. **JavaScript:** Core DOM interactions and event handling.
